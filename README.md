### 1. Склонируйте к себе репозиторий, в котором хранится проект тестового задания, через выполнение команды в терминале
```
git clone https://github.com/Murai-s/avito-qa-task.git
```
### 2. Для запуска тестов вам понадобятся Java (JDK) и Maven. Убедитесь, что они установлены на вашем компьютере. В командной строке/терминале выполните команды:
```
java -version
mvn -version
```
Если Java не установлена: скачайте и установите JDK (версии 17 или выше), например [тут](https://www.oracle.com/java/technologies/downloads/).  
Если Maven не установлен: скачайте его с [официального сайта](https://maven.apache.org/download.cgi) и следуйте инструкции по установке.

### 3. Через терминал перейдите в корневую директорию проекта (туда, где лежит файл pom.xml), выполнив команду:
```
cd /путь/до/папки/avito-qa-task
```

### 4. Загрузите необходимые зависимости выполнив команду:
```
mvn clean dependency:resolve
```

### 5. Запустите тесты, выполнив команду:
```
mvn test
```

После выполнения команды в терминале отобразится отчет о прохождении тестов (количество успешных, упавших и пропущенных кейсов).
